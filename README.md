# Secure Login System

## Overview

This project is a secure web-based login system developed using Python, Flask, SQLite, and bcrypt. It provides user registration, authentication, password hashing, and session management to protect user accounts.

## Features

* User Registration
* User Login and Logout
* Password Hashing using bcrypt
* Session Management
* Input Validation
* SQL Injection Protection using Parameterized Queries

## Technologies Used

* Python
* Flask
* SQLite
* bcrypt
